# Pixiv-Web

*p站爬虫加展示web*

#### 功能
* 展示Pixiv三榜排名前五十的图片
* 拥有收藏图片、下载图片、访问Pixiv原链接功能
* 使用json文件进行本地数据存储
* 可以通过id搜素Pixiv的图片并添加到本地，也可以通过关键字搜索本地图片
* 每天凌晨自动爬取图片，也可手动点击刷新按钮更新图片
